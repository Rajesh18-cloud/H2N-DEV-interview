Parsing XML to JSON:

Python script to convert your XML into a Python dictionary and then print it as JSON:

import xml.etree.ElementTree as ET
import json

# XML data
xml_data = """
<Order>
    <OrderID>1004</OrderID>
    <Customer>
        <CustomerID>4</CustomerID>
        <Name>Emily Johnson</Name>
    </Customer>
    <OrderDate>2024-10-15T13:30:00</OrderDate>
    <Products>
        <Product>
            <ProductID>104</ProductID>
            <Name>USB Hub</Name>
            <Quantity>3</Quantity>
            <Price>15.00</Price>
        </Product>
    </Products>
    <TotalAmount>45.00</TotalAmount>
</Order>
"""

# Function to convert XML to dictionary
def xml_to_dict(element):
    if len(element) == 0:
        return element.text
    return {child.tag: xml_to_dict(child) for child in element}

# Parse XML data
root = ET.fromstring(xml_data)
order_dict = xml_to_dict(root)

# Convert dictionary to JSON
order_json = json.dumps(order_dict, indent=4)

# Print JSON
print(order_json)

Running this script will parse the XML, convert it into a dictionary, and then convert that dictionary into JSON format. The JSON output will look like this:
